# 寒假成果報告

這份檔案包含整個寒假活動的記錄。只需打開一個 HTML 檔案，即可在瀏覽器中查看完整內容。

## 使用說明

### 1. 下載資料夾

**最簡單的方法：**
- 在 GitHub 頁面點「Code」，選「Download ZIP」下載後解壓。

**使用命令列（如果懂的話）：**
```bash
git clone https://github.com/andychou978/Andy-s-winter-report-2026.git
cd Andy-s-winter-report-2026
```

**如何打開命令列（Windows）：**
1. 按鍵盤上的 Win 鍵
2. 輸入 `cmd`
3. 按 Enter 打開黑色視窗

### 2. 打開報告

在資料夾中找到檔案：`寒假成果報告.html`（網頁圖示），雙擊打開。

![HTML 檔案圖示範例](screenshot.png)

- 或在瀏覽器內手動輸入檔案路徑打開。
- 不需要 Python 或其他軟體。

### 3. 分享

直接寄送 `寒假成果報告.html` 檔案即可，所有內容已內嵌其中。
